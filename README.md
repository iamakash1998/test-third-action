# Node.js Calculator App

A simple command-line calculator built with Node.js.

## Installation

1. Open a terminal in the project folder.
2. Run `npm install` if you want to initialize the project dependencies.

## Usage

Run the app with the desired operation and two numbers:

```bash
node app.js add 5 3
node app.js sub 10 4
node app.js mul 6 7
node app.js div 12 4
```

## Supported Operations

- `add` - addition
- `sub` - subtraction
- `mul` - multiplication
- `div` - division

## Help

```bash
npm run help
```
